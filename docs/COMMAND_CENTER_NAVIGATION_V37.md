# Command Center Navigation v37

This release turns the growing GoalOS AGIJobManager Ascension website into a coherent public product surface.

It adds:

- a flagship `command-center.html` front door;
- curated user paths for first-time visitors, executives, builders, operators, and risk/legal reviewers;
- a complete page catalog;
- client-side filtering without network requests or storage;
- a compact universal navigation layer injected into generated pages;
- source and post-build tests for navigation completeness.

Public-safe posture remains unchanged: no accounts, no forms, no analytics, no cookies, no wallet connection, no transaction broadcast, no user data wanted.
