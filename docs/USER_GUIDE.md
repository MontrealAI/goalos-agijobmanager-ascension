# User Guide

GoalOS AGIJobManager Ascension is easiest to use as a guided public website: start at the homepage, choose a route, run a browser-local proof demonstration, and read the boundary before interpreting any output.

## Two-minute path

1. Open the production site.
2. Click **Experience Concierge**.
3. Choose **I want proof** or **I want the legal/privacy boundary**.
4. Export or copy only public-safe sample receipts.

## What outputs mean

Evidence Dockets, ProofBundles, SettlementReceipts, RSIDossiers, and GovernanceDockets are demonstration artifacts. They do not certify external factual correctness, authorize production action, move funds, or prove achieved AGI/ASI.

## Boundaries

No account, forms, analytics, cookies, public wallet connection, token approval, transaction broadcast, funds movement, user data collection, or production authority is required or wanted.
