# Experience Hub v37

v37 completes the public navigation layer for GoalOS AGIJobManager Ascension.

It adds:

- a rewritten front door (`index.html`)
- a complete `experience-hub.html`
- searchable demo catalog
- persona-based guided paths
- universal floating site guide across pages
- public-safe NavigationReceipt export

The implementation preserves the Data-Zero posture: no accounts, forms, analytics, cookies, wallet connection, network requests to third parties, token route, transaction broadcast, or user data collection.
