# GitHub Web UI Guide · v29 Proof-Carrying Artifact Passport

1. Download and unzip the v29 overlay.
2. Open `https://github.com/MontrealAI/goalos-agijobmanager-ascension`.
3. Click **Add file → Upload files**.
4. Drag the contents of the overlay into GitHub.
5. Commit to `main` with: `Add Proof-Carrying Artifact Passport v29`.
6. Open **Actions**.
7. Run **GoalOS AGIJobManager Ascension Proof-Carrying Artifact Passport Publisher v29**.
8. Use `deploy_pages: true`, `commit_generated_source: true`, `run_live_factual_check: false`, `strict_live_factual_check: false`.
9. Verify `/proof-carrying-artifact-passport.html` and `/proof-carrying-artifact-passport-demo.json`.
