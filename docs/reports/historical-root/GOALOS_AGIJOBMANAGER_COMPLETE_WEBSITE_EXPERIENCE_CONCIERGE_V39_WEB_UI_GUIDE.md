# v39 GitHub Web UI Guide

1. Download and unzip the v39 overlay.
2. Open the GitHub repository.
3. Click **Add file → Upload files**.
4. Drag the **contents** of the unzipped overlay into GitHub.
5. Commit directly to `main` with `Add Experience Concierge v39`.
6. Open **Actions**.
7. Run **GoalOS AGIJobManager Ascension Complete Website Publisher v39**.
8. Use `deploy_pages: true`, `commit_generated_source: true`, `run_live_factual_check: false`, and `strict_live_factual_check: false`.
9. Verify `/`, `/experience-concierge.html`, `/experience-hub.html`, `/command-center.html`, `/proof-settlement-lifecycle.html`, and `/production-url.json`.
