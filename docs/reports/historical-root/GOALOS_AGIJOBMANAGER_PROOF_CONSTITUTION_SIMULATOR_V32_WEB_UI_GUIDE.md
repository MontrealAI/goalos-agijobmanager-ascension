# Proof Constitution Simulator v32 · GitHub Web UI Guide

1. Download and unzip the v32 overlay.
2. Open `https://github.com/MontrealAI/goalos-agijobmanager-ascension`.
3. Click **Add file → Upload files**.
4. Drag the **contents** of the unzipped overlay into GitHub.
5. Commit directly to `main` with:

   `Add Proof Constitution Simulator v32`

6. Open **Actions**.
7. Run:

   `GoalOS AGIJobManager Ascension Proof Constitution Publisher v32`

8. Use:

   - `deploy_pages: true`
   - `commit_generated_source: true`
   - `run_live_factual_check: false`
   - `strict_live_factual_check: false`

9. After the run is green, verify:

   - `/proof-constitution-simulator.html`
   - `/proof-constitution-simulator-demo.json`
   - `/ascension-flight-deck.html`
   - `/production-url.json`
