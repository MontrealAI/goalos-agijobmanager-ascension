# Trust Equation Simulator v35 · GitHub Web UI Guide

1. Download and unzip the v35 overlay.
2. Open `https://github.com/MontrealAI/goalos-agijobmanager-ascension`.
3. Click **Add file → Upload files**.
4. Drag the **contents** of the unzipped overlay into GitHub.
5. Commit directly to `main` with: `Add Trust Equation Simulator v35`.
6. Go to **Actions**.
7. Run **GoalOS AGIJobManager Ascension Trust Equation Publisher v35**.
8. Use `deploy_pages: true`, `commit_generated_source: true`, `run_live_factual_check: false`, `strict_live_factual_check: false`.
9. Verify `/trust-equation-simulator.html` and `/trust-equation-simulator-demo.json`.
