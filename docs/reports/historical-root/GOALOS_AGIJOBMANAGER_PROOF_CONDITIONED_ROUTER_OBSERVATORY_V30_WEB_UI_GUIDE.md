1. Download and unzip `goalos-agijobmanager-ascension-proof-conditioned-router-observatory-v30-overlay.zip`.
2. Open `https://github.com/MontrealAI/goalos-agijobmanager-ascension`.
3. Click **Add file → Upload files**.
4. Drag the contents of the unzipped overlay into GitHub.
5. Commit directly to `main` with: `Add Proof-Conditioned Router Observatory v30`.
6. Open **Actions**.
7. Run **GoalOS AGIJobManager Ascension Proof-Conditioned Router Observatory Publisher v30**.
8. Use `deploy_pages: true`, `commit_generated_source: true`, `run_live_factual_check: false`, `strict_live_factual_check: false`.
9. Verify `/proof-conditioned-router-observatory.html` and `/proof-conditioned-router-observatory-demo.json`.
