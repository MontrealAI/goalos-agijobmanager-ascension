# Proof-Settlement Lifecycle Console v36 · GitHub Web UI Guide

1. Download the v36 overlay ZIP.
2. Unzip it.
3. Open the repository on GitHub.
4. Click **Add file → Upload files**.
5. Drag the contents of the unzipped overlay into GitHub.
6. Commit directly to `main` with: `Add Proof-Settlement Lifecycle Console v36`.
7. Go to **Actions**.
8. Run **GoalOS AGIJobManager Ascension Proof-Settlement Publisher v36**.
9. Keep `deploy_pages = true`, `commit_generated_source = true`, and live factual checks off unless an RPC secret is configured.
10. Verify `/proof-settlement-lifecycle.html` and `/proof-settlement-lifecycle-demo.json`.
