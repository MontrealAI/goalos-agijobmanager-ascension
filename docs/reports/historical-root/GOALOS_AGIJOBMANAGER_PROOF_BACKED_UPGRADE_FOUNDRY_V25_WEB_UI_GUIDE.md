# GitHub Web UI Guide — Proof-Backed Upgrade Foundry v25

1. Download and unzip the v25 overlay.
2. Open https://github.com/MontrealAI/goalos-agijobmanager-ascension
3. Click **Add file → Upload files**.
4. Drag the contents of the unzipped overlay into GitHub.
5. Commit directly to `main` with: `Add Proof-Backed Upgrade Foundry v25`.
6. Go to **Actions**.
7. Run **GoalOS AGIJobManager Ascension Proof-Backed Upgrade Foundry Publisher v25**.
8. Use `deploy_pages: true`, `commit_generated_source: true`, `run_live_factual_check: false`, `strict_live_factual_check: false`.
9. Verify `/proof-backed-upgrade-foundry.html` and `/proof-backed-upgrade-foundry-demo.json`.
