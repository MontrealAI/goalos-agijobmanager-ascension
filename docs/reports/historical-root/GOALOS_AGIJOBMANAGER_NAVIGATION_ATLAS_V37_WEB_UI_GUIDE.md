# v37 GitHub Web UI Guide

1. Download the v37 overlay ZIP.
2. Unzip it on your computer.
3. Open `https://github.com/MontrealAI/goalos-agijobmanager-ascension`.
4. Click **Add file → Upload files**.
5. Drag the **contents** of the unzipped overlay into GitHub.
6. Commit directly to `main` with: `Add Navigation Atlas v37`.
7. Go to **Actions**.
8. Run **GoalOS AGIJobManager Ascension Navigation Atlas Publisher v37**.
9. Keep `deploy_pages: true`, `commit_generated_source: true`, `run_live_factual_check: false`, and `strict_live_factual_check: false`.
10. Verify `/`, `/experience-atlas.html`, `/site-navigation-v37.json`, and `/production-url.json`.
