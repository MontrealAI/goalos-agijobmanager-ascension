# Until-DONE Mission Control v33

Until-DONE Mission Control is a browser-local public demonstration of the GoalOS Mission OS state machine.

It teaches the core product law:

- set the objective
- convert it into a mission contract
- run bounded workstreams
- verify claims
- produce an Evidence Docket
- emit a governed decision state
- prepare an Action Graph
- write Chronicle memory
- stop at human review

The page deliberately includes loopback behavior. If evidence is missing, contradictions remain unresolved, the risk ledger is incomplete, or rollback is not ready, the run does not pretend to be done. It returns to the earliest state that can fix the defect.

Public-safety posture:

- no user data wanted
- no wallet
- no forms
- no analytics
- no cookies
- no localStorage/sessionStorage
- no network request
- no production authority
