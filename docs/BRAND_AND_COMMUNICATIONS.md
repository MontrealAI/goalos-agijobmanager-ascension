# Brand and Communications

GoalOS AGIJobManager Ascension should feel like a credible public proof institution: precise, calm, useful, and proof-bound.

## Core message

> A model can answer. An agent can act. An institution must prove.

The public site should communicate one simple idea: AI output becomes institutional work only when it survives proof, validation, settlement, Chronicle, and human authority boundaries.

## Preferred language

Use: public-safe, proof-bound, evidence room, Evidence Docket, ProofBundle, Governed Decision State, SettlementReceipt, Chronicle, verifier mesh, claim boundary, replay-ready, rollback-ready, human-reviewed, browser-local, default-deny, proof-aligned source.

Avoid: guaranteed, risk-free, investment, profit, yield, buy, sell, moon, price, liquidity, achieved AGI, achieved ASI, certified, audited, externally validated, SOTA, autonomous sovereignty, safe autonomy, or production-ready authority unless an exact Evidence Docket supports the claim.

## Public communication spine

1. Start with the objective.
2. Show the proof path.
3. Show what passed and what is still bounded.
4. Make the public/private boundary visible.
5. Offer a simple next step.

## Social preview line

Proof-settled machine work under human authority.

## Press-safe description

GoalOS AGIJobManager Ascension is a public-safe proof-settlement website and repository for exploring how autonomous AI work can be structured as Evidence Dockets, ProofBundles, governed decision states, settlement receipts, and Chronicle entries without collecting user data or granting public production authority.

## Boundaries to repeat

- Public demos are local simulations.
- No user data wanted.
- No public wallet connection.
- No token offer or brokerage.
- No external audit or certification claim.
- Stronger empirical claims require real tasks, baselines, replay logs, validators, cost/risk ledgers, delayed outcomes, and independent reproduction.
