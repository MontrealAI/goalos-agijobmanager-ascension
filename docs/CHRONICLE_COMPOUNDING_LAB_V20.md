# Chronicle Compounding Lab v20

Status: production-ready browser-local demonstration.

This page demonstrates the GoalOS memory rule: **No Chronicle entry, no institutional memory.**

The demo lets users choose a public-safe mission, run a proof cycle, inspect gates, watch proof debt change, see accepted work become reusable capability, and export a public-safe Evidence Docket.

## Why this page exists

Previous public demonstrations showed objective-to-proof-to-decision and Selection Gate behavior. v20 adds the compounding layer: accepted proof becomes Chronicle memory, and Chronicle memory makes the next mission easier. Rejected work remains visible as a warning but cannot become authority.

## Public safety

The page is local-only and does not use accounts, cookies, analytics, browser storage, wallet primitives, network requests, token routing, transaction broadcasting, or production authority. It explicitly instructs users not to enter personal, confidential, regulated, wallet, seed phrase, private-key, customer, or proprietary information.
