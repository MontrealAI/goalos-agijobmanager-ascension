# Data-Zero Privacy Policy

_Last updated: 2026-06-27_

## Plain-English position

This project is designed to avoid user data. We do not want personal data. We do not need personal data. Do not submit personal data. Do not send personal, confidential, regulated, customer, health, financial, biometric, child, employment, government-ID, secret, private-key, wallet seed, privileged, or proprietary data through this website, repository, issue tracker, pull request, wallet field, transaction memo, URI, or Evidence Docket.

## What the site intentionally does not do

- no accounts;
- no login;
- no forms;
- no analytics;
- no advertising pixels;
- no cookies;
- no fingerprinting;
- no customer database;
- no support chat;
- no mailing list;
- no server-side application controlled by this repository;
- no private-key storage;
- no custody;
- no collection of wallet secrets.

## Browser-local operation

The read-only chamber runs in the visitor's browser and creates unsigned JSON for review. If a visitor downloads or copies that JSON, the visitor controls that file and is solely responsible for its contents. The project does not receive it.

The separated Expert Console can interact with the visitor's wallet only after explicit user action. Wallet addresses, RPC requests, approvals, gas estimates, and transactions are handled by the user's wallet, selected RPC provider, Ethereum Mainnet, and other independent third parties. The project does not control those third parties and does not receive wallet secrets.

## Third-party hosting

The website is published through GitHub Pages. GitHub may process technical data under GitHub's own policies. Visitors who do not accept GitHub's hosting and privacy posture should not access the website.

## User responsibility

Users and operators are solely responsible for ensuring that anything they type, upload, publish, sign, broadcast, reference, or link is lawful, public-safe, non-confidential, properly licensed, and free of personal or regulated data.

## Deletion requests

Because the project intentionally does not collect user data, we generally have no account or user-data record to access, correct, export, or delete. Data a user voluntarily publishes to GitHub, a wallet, an RPC provider, IPFS, Etherscan, Ethereum Mainnet, X, or any other third-party system is controlled by that third party or by the user, and blockchain data may be public and irreversible.
