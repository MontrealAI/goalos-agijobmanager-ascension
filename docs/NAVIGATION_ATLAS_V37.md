# Navigation Atlas v37

GoalOS AGIJobManager Ascension now has a compact primary navigation and a complete Experience Atlas.

The design goal is simple: keep every page and demo, but make the system navigable.

## User paths

- Start: first-time and non-technical users.
- Experience Atlas: complete route map and guided journeys.
- Flight Deck: guided demo launcher with live previews.
- Mission Control: Until-DONE state machine.
- Evidence: Evidence Docket Composer and claim boundary.
- Architecture: implementation and authority separation.
- Legal: Data-Zero, token boundary, and third-party responsibility.

## Public-safe posture

The navigation and atlas remain browser-local:

- no account
- no forms
- no wallet
- no network request beyond same-origin static JSON
- no analytics
- no cookies
- no localStorage/sessionStorage
- no user data wanted
