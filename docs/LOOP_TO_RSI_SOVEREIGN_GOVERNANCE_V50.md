# From Loop to RSI · Sovereign Governance v50

`/loop-to-rsi-sovereign-governance.html` is a browser-local demonstration of the transition from long-running agent loops into deterministic RSI governance.

It demonstrates:

- contract-first loops becoming stateful governance;
- TARGET → EMIT → FILTER → ATLAS → TEST-PLAN → EVAL → INSERT → PROMOTE;
- Search control without outcome authority;
- Evidence Contact Index discipline;
- Move-37 breakthrough handling;
- reproduction, stress tests, persistence gates, and dossier packaging;
- public-safe `RSIDossier` export.

Boundary: this is an architecture demonstration. It does not claim achieved AGI, ASI, empirical SOTA, external audit completion, production certification, autonomous sovereignty, or live invention authority. Public demos remain no-wallet, no-network, no-storage, and no-user-data-wanted.
