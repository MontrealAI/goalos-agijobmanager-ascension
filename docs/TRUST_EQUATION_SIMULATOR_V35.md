# Trust Equation Simulator v35

The Trust Equation Simulator is a browser-local public demonstration of the GoalOS law:

```text
AI Work = Output × Proof × Validation × Settlement × Reuse
If proof is zero, trusted institutional work is zero.
```

The page helps non-technical and advanced users see why output quality is insufficient without proof, validation, settlement readiness, reuse boundaries, rollback, and claim discipline.

Public-safety posture:

- no account
- no form submission
- no wallet connection
- no network request
- no analytics
- no cookies
- no localStorage/sessionStorage
- no user data wanted
