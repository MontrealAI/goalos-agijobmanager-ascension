# Loop → RSI Control Room v51

Route: `/loop-to-rsi-control-room.html`

This public-safe demonstration shows how long-running agent loops become governable only when they are converted into deterministic RSI state: mission contract, separated roles, virtual disk state, trace reading, ECI, baseline comparison, Move-37 handling, replay, dossier packaging, and human review boundary.

## What users can do

- choose a public-safe mandate preset;
- run the RSI cycle one gate at a time;
- inject a high-novelty Move-37 candidate;
- restart from manifest;
- quarantine unsafe or unsupported candidates;
- inspect virtual `contract.json`, `progress.md`, `rsi_state.json`, and `dossier.md` state;
- download an `RSIDossier` JSON;
- copy a review brief.

## Public-safe boundary

No account, no form, no wallet, no token approval, no network request, no analytics, no cookies, no browser storage, no transaction broadcast, no production authority, and no user data wanted.

## Claim boundary

This page is a browser-local demonstration. It does not claim achieved RSI, AGI, ASI, SOTA, external audit, or production authority. Stronger claims require real tasks, baselines, executed evidence, replayable ProofBundles, cost/risk ledgers, validator reports, delayed outcomes, and independent review.
