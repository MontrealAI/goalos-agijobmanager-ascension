# v41 GitHub Web UI Guide

1. Download the v41 overlay ZIP.
2. Unzip it.
3. Open the repository on GitHub.
4. Click **Add file → Upload files**.
5. Drag the contents of the unzipped overlay into GitHub.
6. Commit directly to `main` with `Add Navigation Command Final v41`.
7. Go to **Actions**.
8. Run **GoalOS AGIJobManager Ascension Navigation Command Final Publisher v41**.
9. Keep `deploy_pages` and `commit_generated_source` checked.
10. Leave live factual checks off unless `ETHEREUM_RPC_URL` is configured.
