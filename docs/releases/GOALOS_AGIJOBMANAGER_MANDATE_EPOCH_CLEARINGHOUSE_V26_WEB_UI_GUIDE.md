# MandateEpoch Clearinghouse v26 — GitHub Web UI Guide

1. Download and unzip the v26 overlay.
2. Open `https://github.com/MontrealAI/goalos-agijobmanager-ascension`.
3. Click **Add file → Upload files**.
4. Drag the contents of the unzipped overlay into GitHub.
5. Commit directly to `main` with: `Add MandateEpoch Clearinghouse v26`.
6. Go to **Actions**.
7. Run **GoalOS AGIJobManager Ascension MandateEpoch Clearinghouse Publisher v26**.
8. Use `deploy_pages: true`, `commit_generated_source: true`, `run_live_factual_check: false`, and `strict_live_factual_check: false`.

After the run is green, open:

- `https://montrealai.github.io/goalos-agijobmanager-ascension/mandate-epoch-clearinghouse.html`
- `https://montrealai.github.io/goalos-agijobmanager-ascension/mandate-epoch-clearinghouse-demo.json`
