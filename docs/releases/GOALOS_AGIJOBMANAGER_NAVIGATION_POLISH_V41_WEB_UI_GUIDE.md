# Navigation Polish v41 · GitHub Web UI Guide

1. Download the v41 overlay ZIP.
2. Unzip it.
3. Open the GitHub repository.
4. Click **Add file → Upload files**.
5. Drag the contents of the unzipped overlay into GitHub.
6. Commit directly to `main` with `Add Navigation Polish v41`.
7. Go to **Actions**.
8. Run **GoalOS AGIJobManager Ascension Navigation Polish Publisher v41**.
9. Keep deploy Pages and commit generated source enabled.
10. Keep live factual check disabled unless `ETHEREUM_RPC_URL` is configured.
