# v62 — Ask GoalOS Sovereign Question Router

v62 upgrades Ask GoalOS from a simple local question router into a richer public proof concierge.

## Added / improved

- Intent scoring over the canonical route manifest.
- Query expansion for proof, evidence, settlement, loop, RSI, ASI horizon, AGIALPHA, legal, privacy, and route recovery questions.
- Confidence indicator.
- Answer explanation plus route recommendations.
- Explicit navigation only when requested by the visitor.
- Downloadable `GoalOSQuestionRoutingReceipt`.
- Full-page console improvements.
- Floating widget improvements.
- Public-safe verification test and kernel.

## Boundary

No account, no form submission, no public wallet connection, no token approval, no analytics, no cookies, no browser storage, no transaction broadcast, no production authority, and no user data retained.
