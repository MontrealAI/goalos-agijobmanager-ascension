# Repository Map

- `README.md` — public front door.
- `site/` — static public website.
- `data/` — route manifests, catalogs, and demo data contracts.
- `schemas/` — JSON schemas.
- `docs/` — documentation spine, reports, and releases.
- `tools/` — build, verification, static checkers, dynamic runners.
- `tests/` — dependency-zero tests.
- `.github/` — issue/PR templates and Pages publisher.
