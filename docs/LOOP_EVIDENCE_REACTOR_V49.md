# Loop Evidence Reactor v49

Loop Evidence Reactor is a browser-local GoalOS demonstration for long-running agent loops. It shows how a prompt becomes a loop, how a loop becomes a proof process, and how proof-bearing cycles become reviewable institutional memory.

## What it demonstrates

- Contract-first operation.
- Planner / builder / evaluator / trace-reader separation.
- Virtual disk state through `contract.json`, `progress.md`, `log.md`, and `bottleneck.md`.
- Crash and restart from written state.
- Explicit scoring of subjective quality through a rubric.
- Harness deletion when scaffold becomes drag.
- Bottleneck exposure as the next mission.
- `LoopDocket` export.

## Public-safe boundary

The page is local demonstration software. It does not collect user data, connect wallets, approve tokens, switch networks, broadcast transactions, move funds, or grant production authority.

## GoalOS translation

Objective → Mission Contract → Bounded Roles → Trace → Verifier → Evidence Docket → Governed Decision State → Chronicle → Reusable Capability.
