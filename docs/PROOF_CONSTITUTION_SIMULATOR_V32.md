# Proof Constitution Simulator v32

The Proof Constitution Simulator is a browser-local public demonstration of the GoalOS constitutional core.

It turns the doctrine **Aim → Act → Prove → Evolve** into an interactive workbench:

- choose a scenario;
- inspect the protocol objects;
- tune proof gates;
- compute conformance level;
- see whether the system should publish, hold, canary, rollback, or reject;
- export a public-safe ConstitutionReceipt JSON.

The page demonstrates that proof gates are not decoration. A claim, artifact, job, or public page earns authority only when the required object lifecycle is present and the public/private proof boundary is preserved.

Public-safety posture:

- no account;
- no form submission;
- no wallet;
- no analytics;
- no cookies;
- no storage;
- no network request;
- no user data wanted.
