# GoalOS Proof-to-Action Theatre v18

This page is an incremental public demonstration of the core GoalOS idea: set an objective, convert it into a Mission Contract, run until proof gates are complete, emit an Evidence Docket, produce a Governed Decision State, stop at human review, and preserve reusable capability only as a review-ready artifact.

The page is browser-local and public-safe. It uses no account, no form submission, no analytics, no cookies, no localStorage, no sessionStorage, no wallet connection, no token approval, no network request, and no transaction broadcast.

## User path

1. Pick a mission preset.
2. Run until DONE.
3. Compare report-only output with GoalOS proof-to-action flow.
4. Inspect proof gates, Chronicle trace, proof debt, and decision readiness.
5. Download a public-safe Evidence Docket JSON.

## Core doctrine demonstrated

- AI creates output; GoalOS creates proof.
- A report is not the deliverable; the deliverable is a governed decision state.
- No proof, no settlement. No eval, no propagation. No rollback, no release.
- The page makes claim boundaries visible and stops at human review.
