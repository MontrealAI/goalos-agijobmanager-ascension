# Security and Privacy Boundary

[Docs index](README.md)

| Boundary | Public posture |
| --- | --- |
| User data | No user data wanted. |
| Forms | No forms on public demos. |
| Analytics | No analytics. |
| Cookies | No cookies. |
| Browser storage | No localStorage/sessionStorage requirement. |
| Wallets | No public wallet calls. |
| Tokens | No token approvals on public pages. |
| External action | No external action from public pages. |

Public pages may show commitments, summaries, attestations, and receipts. Private prompts, raw traces, customer data, confidential workpapers, and private evaluator notes remain outside the public proof surface.
