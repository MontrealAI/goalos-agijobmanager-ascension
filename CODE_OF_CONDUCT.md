# Code of Conduct

We use a simple respectful collaboration policy.

- Be kind, precise, and constructive.
- Critique claims, code, and evidence; do not attack people.
- Preserve public-safe boundaries and avoid unsupported claims.
- Do not post secrets, personal data, private keys, seed phrases, or confidential workpapers.
- Help new contributors find the safest first click and the relevant documentation.

Maintainers may moderate issues, pull requests, or discussions that violate these expectations.
