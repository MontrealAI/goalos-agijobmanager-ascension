# Code of Conduct

GoalOS AGIJobManager Ascension is a public proof institution. Contributions should be precise, respectful, evidence-oriented, and claim-bounded.

## Expected behavior

- Be direct, civil, and constructive.
- Preserve user safety, privacy, and public/private proof boundaries.
- Separate architecture claims from empirical claims.
- Discuss disagreements through evidence, tests, and reproducible examples.

## Unacceptable behavior

- Harassment, abuse, or targeted insults.
- Requests for secrets, private keys, seed phrases, personal data, customer data, or confidential traces.
- Token promotion, investment solicitation, unsupported claims, or attempts to weaken legal/privacy boundaries.

Maintainers may edit, close, or remove issues and pull requests that violate these expectations.
