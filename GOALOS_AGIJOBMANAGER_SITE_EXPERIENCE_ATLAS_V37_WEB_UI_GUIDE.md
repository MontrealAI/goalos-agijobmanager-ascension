# Site Experience Atlas v37 · GitHub Web UI Guide

1. Download the v37 overlay ZIP.
2. Unzip it locally.
3. Open `https://github.com/MontrealAI/goalos-agijobmanager-ascension`.
4. Click **Add file → Upload files**.
5. Drag the **contents** of the unzipped overlay into GitHub.
6. Commit directly to `main` with: `Add Site Experience Atlas v37`.
7. Go to **Actions**.
8. Run **GoalOS AGIJobManager Ascension Site Experience Atlas Publisher v37**.
9. Keep `deploy_pages: true`, `commit_generated_source: true`, `run_live_factual_check: false`, `strict_live_factual_check: false`.
10. Verify `/`, `/experience-atlas.html`, `/ascension-flight-deck.html`, and `/production-url.json`.
