What the page teaches

A report is not the deliverable.
The decision state is.

The theatre turns a user objective into visible protocol objects: Mission Contract, Claims Matrix, Verifier Report, Risk Ledger, Action Graph, Chronicle Entry, and Capability Package.

For everyone
One button explains the system
Visitors can run the proof cycle, watch why unsupported output is not enough, and download a review-ready docket without technical setup.
For advanced users
Protocol objects are inspectable
The generated JSON exposes the internal objects and claim boundaries so developers can see the architecture behind the demonstration.
For institutions
Authority stops at review
The final state is human review, not automatic execution. The demo shows how proof makes action governable without taking action itself.
