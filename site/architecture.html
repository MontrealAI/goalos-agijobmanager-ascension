The website is not the contract. The console is not a wallet. The Evidence Docket is not a third-party audit claim. Each layer has a narrower authority surface, and Mainnet writes are isolated in the expert console.

Layer 02

Source, schemas, manifests, QA, and GitHub Pages generator for the public institution around the contract.

Layer 03

Read-only chamber

A local browser console that composes unsigned intent and Evidence Docket JSON without wallet permissions.

Layer 04

Evidence room

Public proof summary, non-claims, safety ledger, and review boundaries for readers and institutions.

Layer 05

Expert Mainnet console

The existing advanced console remains separate for operators who understand wallet, token, allowance, and transaction risk.

Layer 06

Human authority

Publication can prepare a decision. It cannot grant authority, certify factual correctness, or authorize funds by itself.
