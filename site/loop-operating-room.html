GoalOS translation

Loop = Mission Contract + State Machine + Evidence Docket

The loop is not hidden context. It is public-safe procedure: what was asked, which roles acted, what changed, what failed, what restarted, what passed, and what remains human.

  1. Objective becomes a contract.
  2. Roles negotiate done criteria.
  3. Work writes artifacts and traces.
  4. Evaluator scores function, craft, originality, and safety.
  5. Verifier blocks unsupported claims.
  6. Chronicle records only accepted state.
