The coordination law

Activity is cheap.
Accepted proof is scarce.

More agents are not automatically better. The useful system is the one that turns model capability into verified work, verified work into reusable capability, and reusable capability into a safer future mission.

AI Work = Output × Proof × Validation × Settlement × Reuse

Maximum coordination = verified useful work per unit of risk, cost, latency, and proof debt.

Operating rule: Only proof-bearing work can advance. Only validated work can settle. Only accepted work can enter Chronicle. Only reusable capability can influence future missions.
