# v42 GitHub Web UI Guide

1. Download and unzip the v42 overlay.
2. Open https://github.com/MontrealAI/goalos-agijobmanager-ascension.
3. Click **Add file → Upload files**.
4. Drag the **contents** of the unzipped overlay into GitHub.
5. Commit directly to `main` with: `Add Institutional Website Finalization v42`.
6. Open **Actions**.
7. Run **GoalOS AGIJobManager Ascension Institutional Website Publisher v42**.
8. Keep `deploy_pages=true`, `commit_generated_source=true`, `run_live_factual_check=false`, and `strict_live_factual_check=false`.
9. Verify the homepage, Start, Concierge, Command Center, Coordination Lab, and production-url.json.
